# Automação de Testes com CodeceptJS + WebDriver  

### Descrição do Projeto

Este projeto foi desenvolvido utilizando CodeceptJS + WebDriver, com foco na automação de testes para aplicações web. Tem por objetivo criar cenários de teste claros, reutilizáveis e de fácil manutenção, garantindo a qualidade e a confiabilidade das funcionalidades testadas.

### Tecnologias e Linguagens Utilizadas

- CodeceptJS - Framework de automação de testes de ponta a ponta. 
- WebDriver -  Tecnologia subjacente usada pelo CodeceptJS para interagir com o navegador.
- JavaScript - Linguagem base para desenvolvimento dos scripts de teste.

Siga os passos abaixo para acessar o projeto e executar os testes.

### Passo 1: Clonar o Projeto

```
git clone https://github.com/eduardoamorimqa/codeceptjs-webdriver.git
```

### Passo 2: Instalar as Dependências

```
npm install
```

### Passo 3: Rodar os Testes

```
npx codeceptjs run --steps
```
